<Intro>

React tích hợp với các công cụ như trình soạn thảo, TypeScript, extension trình duyệt và compiler. Phần này sẽ giúp bạn thiết lập môi trường.

</Intro>

## Thiết lập trình soạn thảo {/*editor-setup*/}

Xem [các trình soạn thảo được khuyến nghị](/learn/editor-setup) và tìm hiểu cách thiết lập chúng để làm việc với React.

## Sử dụng TypeScript {/*using-typescript*/}

TypeScript là cách phổ biến để thêm định nghĩa kiểu vào các codebase JavaScript. [Tìm hiểu cách tích hợp TypeScript vào các dự án React của bạn](/learn/typescript).

## React Developer Tools {/*react-developer-tools*/}

React Developer Tools là extension trình duyệt có thể kiểm tra các React component, chỉnh sửa props và state, và xác định các vấn đề về hiệu suất. Tìm hiểu cách cài đặt nó [tại đây](learn/react-developer-tools).

## React Compiler {/*react-compiler*/}

React Compiler là công cụ tự động tối ưu hóa ứng dụng React của bạn. [Tìm hiểu thêm](/learn/react-compiler).

## Bước tiếp theo {/*next-steps*/}

Hãy đến trang hướng dẫn [Bắt đầu nhanh](/learn) để tìm hiểu các khái niệm React quan trọng nhất mà bạn sẽ gặp hàng ngày.
