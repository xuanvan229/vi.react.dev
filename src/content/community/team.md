<Intro>

Việc phát triển React được dẫn dắt bởi một đội ngũ chuyên trách làm việc toàn thời gian tại Meta. Nó cũng nhận được sự đóng góp từ mọi người trên khắp thế giới.

</Intro>

## React Core {/*react-core*/}

Các thành viên đội ngũ React Core làm việc toàn thời gian trên các API component cốt lõi, engine cung cấp sức mạnh cho React DOM và React Native, React DevTools, và trang web tài liệu React.

Các thành viên hiện tại của đội ngũ React được liệt kê theo thứ tự bảng chữ cái bên dưới.
