<RSC>

Directive được sử dụng trong [React Server Components](/reference/rsc/server-components).

</RSC>

<Intro>

Directive cung cấp các chỉ dẫn cho [các bundler tương thích với React Server Components](/learn/creating-a-react-app#full-stack-frameworks).

</Intro>

---

## Directive mã nguồn {/*source-code-directives*/}

* [`'use client'`](/reference/rsc/use-client) cho phép bạn đánh dấu code nào chạy trên client.
* [`'use server'`](/reference/rsc/use-server) đánh dấu các hàm phía server có thể được gọi từ code phía client.
